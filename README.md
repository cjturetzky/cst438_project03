# Final Furnishings
A storefront app for the sale and purchase of second-hand furniture. 
MVP includes account creation/login, posting listings for furniture, and purchasing listings. 

## Tech Stack
- Platform: Android
- Back-end: Springboot
- Persistence Layer: PostGres SQL
